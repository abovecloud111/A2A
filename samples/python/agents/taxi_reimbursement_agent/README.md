## 打车费报销代理

这个示例使用 Agent Development Kit (ADK) 创建了一个"打车费报销"代理，该代理托管为 A2A 服务器。

这个代理处理员工的打车费报销请求，并根据公司规定审核是否符合报销条件（仅报销晚上9点到次日凌晨5点期间、从中关村资本大厦附近出发的打车费）。如果缺少任何详细信息，代理会返回一个网页表单供客户端（或其用户）填写。在客户端填写表单后，代理将完成任务并批准或拒绝报销请求。

## 功能特点

- 收集上车地点、上车时间、乘车日期和金额信息
- 根据公司政策审核打车费报销资格：
  - 仅报销上车时间为晚上9点到次日凌晨5点期间的打车费
  - 上车地点必须是中关村资本大厦附近（可接受：中关村资本大厦、中关村资本大厦北门、海淀区学院南路）
  - 不报销上车地点为中关村、望京、国贸等其他地方的打车费

## 前提条件

- Python 3.9 或更高版本
- UV
- 访问 LLM 的 API 密钥


## 运行示例

1. 导航到示例目录：
    ```bash
    cd samples/python/agents/taxi_reimbursement_agent
    ```
2. 创建包含 API 密钥的环境文件：
   ```bash
   echo "GOOGLE_API_KEY=your_api_key_here" > .env
   ```

4. 运行代理：
    ```bash
    uv run .
    ```
5. 运行一个客户端应用程序（/samples/python/hosts/README.md）
